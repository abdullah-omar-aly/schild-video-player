
#  Schild-video-player 

Video player web component using HTML Templates , Shadow Dom  and Custom Element 



Installation

```bash
  npm i schild-video-player
```

Import the package
```JavaScript
  import "schild-video-player"
```

Use it
```html
    <schild-video-player height="500px" width="90%">
        <video>
            <source src="">
        </video>
    </schild-video-player>
```
